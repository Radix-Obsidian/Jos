Product: Joy V1 Sales Rep - AI Sales Automation for Voco V2
Version: 1.0

Goal: Autonomous AI sales rep that scans leads, writes personalized outreach,
sends emails + LinkedIn messages, follows up, and closes deals for Voco V2.

Pipeline:
1. Lead scanning via web scraping
2. Lead qualification and scoring via local LLM
3. Personalized outreach generation (email + LinkedIn)
4. Automated message delivery
5. Follow-up sequences (day 3, 7, 14)
6. Closing: book demos (enterprise) or send payment links (self-serve)

Channels: Email + LinkedIn
Lead Source: Web scraping (directories, LinkedIn, industry sites)
Close Actions: Book demo call (enterprise) | Send Stripe payment link (self-serve)

Success Metrics:
- 100+ leads scanned per run
- Personalized messages for each qualified lead
- 3-touch follow-up sequence
- Runs 12hr/day on M1 Air without crashes
- Clean handoff to human closer when demo is booked

Out of Scope: Voice agents, code generation, Viper, Comply, any non-sales product.
